### 최소공배수
2개의 숫자들의 배수중에 가장 작으 공통된 배수

A에 큰수 B에 작은수가 들어와야함.
최대 공약수
```java
public static int gcd(int a, int b) {
    while(b > 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```
